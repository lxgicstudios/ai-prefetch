AI suggests routes and data to prefetch for better UX.

## Install

```bash
npm install -g ai-prefetch
```

## Usage

```bash
npx ai-prefetch ./src/pages/
npx ai-prefetch ./app/routes/
```

```bash
export OPENAI_API_KEY=sk-...
```
